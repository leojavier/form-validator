FAQ

Can i use it for free?

Yes! Aragonite is an opensource project authored by Leo Javier and released under an MIT license.

Can i ask for a new functionality?

Yes, you can submit a ticket through Github at any time. You are welcome to contact me via Email or Twitter for suggestions.

What's new in latest release?

Please take a look of the Changelog.

need to validate a field but the pattern doesn’t exist in the regex list?

Please post a question on stackoverflow and tag it as #aragonite. The right answer will be added to the Regex List so anyone including yourself can use it again in the future.
